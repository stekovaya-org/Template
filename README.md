# Template
This is template for Stekovaya.

## Usage
```
$ make help
Help v1.0
Usage: make [args...]
args:
	No args: Install Stekovaya and run Stekovaya.
	install: Install Stekovaya.
	run    : Run Stekovaya.
	license: Display about license info.
	         Need OPT=legalcode/name
	clean  : Uninstall Stekovaya.
```
